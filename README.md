## Loose steps for dev build
Refer to the tutorial for details.

```
$ [set up python venv]
$ pip install flask
$ export FLASK_ENV=devlopment
$ export FLASK_APP=myapp.py

$ flask init-db
$ flask run

Find the app running on
http://127.0.0.1:5000/
```
